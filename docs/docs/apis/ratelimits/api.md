| /auth/v1/*                                          |                          | none                       |
| /caos.zitadel.auth.api.v1.AuthService/*             |                          | none                       |
| /management/v1/*                                    |                          | 240 request per 1 min      |
| /caos.zitadel.management.api.v1.ManagementService/* |                          | 240 request per 1 min      |
