## TLS mode external

```toml
worker_processes  1;
events {
    worker_connections  1024;
}

http {
    server {
        listen 443;

        ssl_certificate     ssl/certificate.pem;
        ssl_certificate_key ssl/key.pem;
 
        location / {
            grpc_pass grpc://localhost:8080;
            grpc_set_header Host $host:$server_port;
        }
    }
}
```
