## TLS mode external

```bash
https://localhost {
        reverse_proxy h2c://localhost:8080
        tls internal #only non production
}
```

## TLS mode enabled

```bash
https://localhost {
        reverse_proxy https://localhost:8080
        tls internal #only non production
}
```

## TLS mode disabled

```bash
http://localhost {
        reverse_proxy h2c://localhost:8080
}
```
