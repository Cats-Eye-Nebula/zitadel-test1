### Scope

**Authorization**
- Service User with PAT Token

**RBAC**
- Public Endpoint
- Private Endpoint / Route
- Private Scoped Endpoint / Route (read:messages)

**Check token against introspection_endpoint**
- Use roles scope (Don’t use “assert roles on authentication”)
